Markdown Preview Enhanced

Markdown Preview Enhanced

一个很好的vscode插件，让markdown成为写作神器

安装

在vscode编辑器中，ctrl+shift+x打开插件页，搜索markdown-preview-enhanced，点击安装。

markdown基本用法

Markdown是一种通过符号表示格式的方法。

语法高亮

在代码块开始符号后跟语言名称，例如 ```c表示c语言语法高亮。

//有语法高亮的代码块
#include <stdio.h>
int main()
{
    printf("hello world!\n");
    return 0;
}

//没有语法高亮的代码块
#include <stdio.h>
int main()
{
    printf("hello world!\n");
    return 0;
}
