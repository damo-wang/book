# 模块机制

喜欢这个章节，是因为喜欢通过commonJS把内容梳理为模块的感觉

喜欢这个章节，是因为可以用喜欢的c和c++语言来做模块

## commonJS 

创建模块
```javascript
//module.js
exports.hello=function(){
    console.log("Hello World");
};
exports.test=function(){
    console.log("output test");
};
```

```javascript
//main.js
var mymod=require("./module");

mymod.hello();
mymod.test();
```
